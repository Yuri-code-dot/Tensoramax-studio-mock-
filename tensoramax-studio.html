Docs Quick Start
Get started in 5 minutes

Install the TensoraMax CLI and train your first model with a single command. Works from any device — including your phone via Termux.

# Install via pip pip install tensoramax --break-system-packages # Authenticate txb login --token <your-token> # Fine-tune your first model txb train \ --base qwen2.5-0.5b \ --dataset my-data.jsonl \ --lora-r 8 \ --output my-first-model ✓ Training started · Job ID: txb_8f2a19c
